📌 Project Description

This mini project predicts salary based on years of experience using Linear Regression.
The objective is to understand how a machine learning model can learn patterns from data and make predictions.

📁 Dataset Information

Input Feature: Years of Experience

Target Variable: Salary

⚙️ Project Workflow

1 Load the dataset

2 Perform basic data analysis

3 Split data into training and testing sets

4 Train the Linear Regression model

5 Predict salary for given experience

6 Visualize the regression results

Scikit-learn
